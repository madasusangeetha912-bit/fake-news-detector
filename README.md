# 📰 Fake News Detector

An AI-powered Fake News Detection web application built using **Python**, **Machine Learning**, and **Streamlit**. The application analyzes news text and predicts whether it is **REAL** or **FAKE** using a trained machine learning model.

## 🚀 Features

- Detects whether a news article is Real or Fake
- Simple and user-friendly Streamlit interface
- Machine Learning based prediction
- Fast and accurate text classification
- Easy to run and use

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## ▶️ Installation

1. Clone the repository

```bash
git clone https://github.com/madasusangeetha912-bit/fake-news-detector.git
```

2. Move to the project folder

```bash
cd fake-news-detector
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run fake_news_app.py
```

## 📸 Output

- Enter any news article in the text box.
- Click the Predict button.
- The application displays whether the news is **REAL** or **FAKE**.
